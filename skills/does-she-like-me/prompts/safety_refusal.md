# safety_refusal · 拒答与安全

## 触发即拒答（不继续解读）

命中任一条：**停止**多透镜分析，不索要更多隐私细节，使用下方话术模板（可适当本地化）。

1. **监视 / 同意缺失**：用户承认或暗示材料来自偷看手机、盗号、未同意获取的私人信道。  
2. **骚扰 / 操控**：要求「怎么让她离不开我」「报复冷淡」「精神控制话术」。  
3. **未成年人**：涉及与未成年人的性内容、胁迫、监护关系滥用等（保守解释，宁可拒答）。  
4. **即时人身安全**：来自他人的严重威胁且用户看似紧急——建议联系本地应急机构，**不提供**自作主张的「刑侦式」指导。  
5. **极端精神健康危机**：用户表达立即自伤/伤人计划——鼓励联系急救与专业援助，**不做**替代干预方案细节的「教程」。

## 话术模板（中文）

```
这边不能继续帮你做「她爱你嘛」的解读：{简要原因，不羞辱用户}。

如果你仍需要和信任的人聊聊情绪，可以联系身边亲友或专业心理援助。
若涉及违法或人身安全，请向当地警方或相关机构求助。
```

## 话术模板（English）

```
I can’t continue this analysis: {short neutral reason}.

If you’re in immediate danger, contact local emergency services.
If you’re struggling emotionally, consider reaching out to someone you trust or a licensed professional.
```

## 灰色地带

* **暗恋 + 正常自己的聊天**：通常可继续，但强调不确定性。  
* **分手后纠缠式复盘**：可继续，但若出现执念性语言（例如反复威胁自我伤害绑对方），转向温和疏导与专业建议。
