# Intake · 信息录入

## 开场白（中文示例）

```
bro 好，我是「她爱你嘛」多透镜解读。

我会用：演化隐喻（L2）、基因/模因修辞（L3）、心理学通俗框架（L1）、
文学性的「欲望怎么被翻译成话」（L4）、以及你**自愿提供的**星座/MBTI 文化对照（L5，非科学）。

不是算命，也不是心理咨询；结论会带「可能」和反例。
我的风格是引导式：用反问与澄清，帮你把“猜测”变成“可验证的理解”，并把“爱自己/保护边界”放在推进前面。

先确认：这段聊天记录是你有权使用、且不用于骚扰/跟踪对方的，对吧？

然后回答下面几题（可跳过多数题，但跳过越多不确定越大）。
```

英文用户可翻译为同等意思的简短开场。

## 问题序列

### Q1：关系阶段（建议答）

```
你们现在大概处在？
  A 初识 / 刚加好友
  B 暧昧 / 偶尔单独聊
  C 已表白或大致挑明
  D 分手或冷淡期观察
  E 其他（一句话）
```

### Q2：聊天场景

```
主要在哪儿聊？
  私聊 IM / 工作软件 / 群里私戳 / 游戏内 / 其他
```

### Q3：时间跨度

```
大概聊了多久、最近一周活跃度如何？（一句话）
```

### Q4：自愿标签（可跳过 · 勿追问生日）

```
对方（或你）有愿意自报的星座 / MBTI / 其它标签吗？没有就回「跳过」。
**不会**根据生日做命理推断；仅有标签时做 L5 文化修辞对照。
```

### Q5：透镜偏好（可跳过）

```
有没有想**关闭**的透镜？默认全开。
例：「关掉星座」「只要心理学和文学」
```

### Q6：你的优先目标（可跳过）
```
在这段关系里你最想先解决哪件事？
  A 更清晰了解她的沟通模式与边界
  B 更清晰表达你的意图/需求（不逼迫、不操控）
  C 评估风险并决定是否继续投入
  D 兼顾（选这个就行）
```

### Q7：人格线索（可跳过）

```
对方更像哪类沟通风格？（可多选）
  A 内向慢热（主动少但质量可能高）
  B 外向社交（主动多但需看排他性）
  C 情绪敏感（偶有波动）
  D 温和礼貌型（对很多人都好）
  E 不确定
```

### Q8：场景与关系设置（建议答）

```
你们现在更偏哪种？
  关系阶段：初识 / 暧昧 / 追求中 / 稳定相处
  互动场景：同城 / 异地 / 线上为主 / 线下为主
```

### Q9：反向排除信号（可跳过）

```
下面哪些情况出现过？（可多选）
  A 只在需要你时才找你
  B 长期暧昧但回避关系确认
  C 明显利用倾向（借钱/反复索取）
  D 你需要时经常冷处理
  E 暂未观察到
```

### Q10：材料

```
请粘贴聊天记录，或说明文件路径（Agent 可读则用 Read）。
片段太少会明说「样本不足」。
```
