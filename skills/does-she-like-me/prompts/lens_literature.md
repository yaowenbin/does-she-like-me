# lens_literature · 文学与「欲望的翻译」（L4）

## 段首固定标注

`【把聊天当故事：她在扮演什么角色 · L4】文本细读；「她想说什么」可有多种译本，禁止单一真理。`

## 分析提纲

1. **缝隙**

   * 字面意义 vs 可能潜台词：省略、反问、停顿（用「…」表现）、撤回（若用户提到）、表情包**代替**了哪句说不出口的话？

2. **多重译本（至少列 3 种）**

   例：同一句关心 → A 暧昧拉扯 B 朋友式礼貌 C 回避深入的情感挡板 D 情境性内疚补偿  
   每种译本用**一句**说明依据：优先写 `（证据：<一句话说明>：「…」）`，否则写 `（推断；待证实）`。

3. **叙事位置**

   她把自己放在：照顾者 / 玩伴 / 被追求者 / 平等朋友 / 旁观者？  
   文本是否留出**双人故事继续伸展**的空间（约定、共同计划、延续话题）？

4. **互文与梗**

   内部梗、作品引用：可能增加**共同叙事资本**；也可能是纯友谊圈层文化。

## 风格

禁止低俗情色堆砌；禁止长篇摘抄网红语录。  
中文约 200～380 字。
