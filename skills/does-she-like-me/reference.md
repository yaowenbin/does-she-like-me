# 她爱你嘛 · reference（术语与误判速查）

本文件供 Agent 与用户共查；分析时**以聊天记录原句为准**，术语仅辅助表达。

---

## 《自私的基因》在本 Skill 中的用法（防望文生义）

理查德·道金斯用语：**复制子（replicator）**、**载体（vehicle）**、**延伸表现型** 等，在本 Skill 中仅作**修辞与思想实验**：讨论「话术模板是否在复制」「偏离习惯的真诚披露是否信息量大」。**不是**基因检测，**不是**「她基因里写着不爱你」。

---

## 演化隐喻常用词（L2）

| 词 | 允许用法 | 禁止用法 |
| -- | -------- | -------- |
| 成本信号 / costly signal | 「持续投入注意力**可能**是高成本信号的一种解读」 | 「成本高所以她爱你」 |
| 互惠 / 博弈 | 「双向提问**像**互惠；也可能是客气」 | 「博弈论算出她赢了你输了所以没爱」 |
| 性选择 | 「科普框架：展示与选择在物种层面的叙事」 | 「达尔文说她要选你」 |

---

## 心理学标签（L1，非诊断）

依恋风格（安全/焦虑/回避/混乱）仅描述**沟通模式谱系**；同一句话可被不同风格解释。不在聊天不足时给对方贴「回避型人格障碍」等诊断。

爱情三角（亲密·激情·承诺）：纯文本对**激情**推断力弱，主要落**亲密与承诺的语言线索**。

---

## 常见误判（写给用户）

| 现象 | 友好解释 | 需警惕的过度解读 |
| ---- | -------- | ------------------ |
| 秒回 | 习惯、空闲、礼貌 | 「一定上头」 |
| 已读慢回 | 忙、想措辞、躲避压力 | 「一定不爱」 |
| 表情包多 | 轻松氛围、防御性诙谐 | 「打情骂俏铁证」 |
| 关心饮食作息 | 文化里常见关怀 | 「唯一的爱意证明」 |
| 短句「嗯嗯」 | 累、敷衍、内向都可 | 「冷暴力」单线定罪 |

---

## 星座模块（L5）

占星**不是科学**。仅当用户**自愿提供**标签时，用作「她可能熟悉的自我叙事语言」的**修辞对照**，且每条须回落到聊天原句验证。不用水逆解释关系；不用某星座「一定渣/专一」。

---

## 证据标注约定

* `（证据：「……」）` — 直接引用用户材料。
* `（推断；待证实）` — 无直接句支撑时的推测，须写明可如何验证或推翻。
