# 她爱你嘛 · reference（术语与误判速查）

本文件供 Agent 与用户共查；分析时**以聊天记录原句为准**，术语仅辅助表达。

---

## 《自私的基因》在本 Skill 中的用法（防望文生义）

理查德·道金斯用语：**复制子（replicator）**、**载体（vehicle）**、**延伸表现型** 等，在本 Skill 中仅作**修辞与思想实验**：讨论「话术模板是否在复制」「偏离习惯的真诚披露是否信息量大」。**不是**基因检测，**不是**「她基因里写着不爱你」。

---

## 演化隐喻常用词（L2）

| 词 | 允许用法 | 禁止用法 |
| -- | -------- | -------- |
| 成本信号 / costly signal | 「持续投入注意力**可能**是高成本信号的一种解读」 | 「成本高所以她爱你」 |
| 互惠 / 博弈 | 「双向提问**像**互惠；也可能是客气」 | 「博弈论算出她赢了你输了所以没爱」 |
| 性选择 | 「科普框架：展示与选择在物种层面的叙事」 | 「达尔文说她要选你」 |

## 心理学标签（L1，非诊断）

依恋风格（安全/焦虑/回避/混乱）仅描述**沟通模式谱系**；同一句话可被不同风格解释。不在聊天不足时给对方贴「回避型人格障碍」等诊断。

爱情三角（亲密·激情·承诺）：纯文本对**激情**推断力弱，主要落**亲密与承诺的语言线索**。

---

## 常见误判（写给用户）

| 现象 | 友好解释 | 需警惕的过度解读 |
| ---- | -------- | ------------------ |
| 秒回 | 习惯、空闲、礼貌 | 「一定上头」 |
| 已读慢回 | 忙、想措辞、躲避压力 | 「一定不爱」 |
| 表情包多 | 轻松氛围、防御性诙谐 | 「打情骂俏铁证」 |
| 关心饮食作息 | 文化里常见关怀 | 「唯一的爱意证明」 |
| 短句「嗯嗯」 | 累、敷衍、内向都可 | 「冷暴力」单线定罪 |

## 星座模块（L5）

占星**不是科学**。仅当用户**自愿提供**标签时，用作「她可能熟悉的自我叙事语言」的**修辞对照**，且每条须回落到聊天原句验证。不用水逆解释关系；不用某星座「一定渣/专一」。

---

## 证据标注约定

* `（证据：<一句话说明>：「……」）` — 直接引用用户材料，并补一句“这句对应的信号是什么”。
* `（证据：「……」）` — 允许省略一句话说明（表示仅有引文、缺少进一步解释）。
* `（推断；待证实）` — 无直接句支撑时的推测，须写明可如何验证或推翻。

---

## 引导型沟通底色（反操控/反伤害）
本 Skill 的“顾问口吻”不是下命令，而是帮助用户把判断建立在证据与边界上。

硬性写法约束：
1. **优先用问题与复核**：当关键点不确定时，优先给 1 个“澄清/复核问题”（可随时收回），再给解释与可观察线索。
2. **建议必须留余地**：下一步沟通/观察要“可退出、可拒绝”，不得暗示控制、威胁、报复或情绪绑架。
3. **以爱自己收束**：即使分析偏低/不明，也要提醒用户如何保护自尊与情绪（例如降低投入节奏、坚持不纠缠、设截止线）。
4. **避免刻板印象**：凡涉及性别差异、人格标签等，只能表达为“可能的沟通偏好/模式”，且必须回落到聊天原句验证；禁止把推测当成结论。

## 恋爱可得性（L6，防误用）

**定义（复述）**：可得性 ≈ 他人在关系中对你形成的「是否相对好接近、成本与风险是否可控、收益是否看起来稳」的主观体感，再叠客观匹配门槛；是**多维加权**框架，**不是**颜值排名，**不是**「好追/难追」羞辱。

**写给分析师（专业）**：可用演化、博弈、社会、神经隐喻等组织证据，但必须绑定引文或标注样本不足；禁止伪精确打分。

**写给用户（通俗）**：用人话短句说明「对方眼里可能的主观感」或「为何材料不够判断」；避免术语轰炸——术语留在 L6 专业小节即可。

**常见误判**：把「回复快」直接等同于「可得性高」；把「委婉拒绝」当成养鱼；把单条聊天当全网口碑。
